# 📝 Todo App на React

Простое и удобное приложение для управления задачами (Todo list), написанное на React с использованием хуков. Позволяет добавлять, отмечать выполненные и удалять задачи.

## 🚀 Функциональность

- Добавление новых задач
- Отметка о выполнении (чекбокс)
- Удаление задачи
- Переход к первой невыполненной задаче
- Счётчик выполненных и оставшихся задач
- Сохранение задач в `localStorage`

## 🛠 Технологии

- **React** (функциональные компоненты + хуки)
- **CSS** 
- **LocalStorage** для персистентности данных
